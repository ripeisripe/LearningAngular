API

Users

  • GET /api/users : liste par id
  • GET /api/users/all : liste par nom
  • GET /api/users/admins : liste les admins
  • GET /api/users/filter?x=y : filtre les users
  • PUT /api/users/:id {data} : update user
  • POST /api/users {data} : crée un nouveau user (email unique)

Topics

  • GET /api/topics : liste par id
  • GET /api/topics/:id : trouve id
  • GET /api/topics/filter?x=y : filtre user, title, content, comment par LIKE - seulement user pour l'instant
  • POST /api/topics {data} : crée un nouveau Topic
  • PUT /api/topics/:id/tags/ [tags] : tag le topic

Comments

  • GET /api/comments : liste par id
  • GET /api/comments/:id : trouve id
  • GET /api/comments/filter?x=y : filtre content par LIKE
  • POST /api/topics {data} : crée un nouveau Topic
  • PUT /api/comments/:id/ {"content":"...", tags:[]} : edit le comment et les tags
